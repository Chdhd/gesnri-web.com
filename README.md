# GeoEnergy Space Nexus Research Institute — Website

This repository contains the public website for the **GeoEnergy Space Nexus
Research Institute (GESNRI)**, built as a static site for publication via
**GitHub Pages**.

## Structure

```
.
├── index.html        # Home — overview of the three pillars, mission & vision
├── projects.html      # Three pillars: six project areas in detail
├── about.html         # Mission, vision, core values, and strategic goals
└── assets/
    ├── style.css       # Shared stylesheet
    ├── main.js         # Navigation behavior
    └── img/
        └── logo.jpg    # Institute logo
```

## Publishing with GitHub Pages

1. Push this repository to GitHub.
2. In the repository, go to **Settings → Pages**.
3. Under **Build and deployment**, set **Source** to "Deploy from a branch".
4. Select the branch (e.g. `main`) and root folder (`/`), then save.
5. The site will be published at `https://<your-username>.github.io/<repo-name>/`.

## Content

The site content is drawn from the Institute's foundational documents:

- **Three Pillars Projects** — Pillar I (Space &amp; Spatial Intelligence: Space
  Data Source, Spatial Data Collection), Pillar II (Climate, Energy &amp;
  Humanitarian Impact: Humanitarian Data Source, Smart-Climate Integration,
  Sustainable Energy Development), and Pillar III (Space Exploration &amp;
  Innovation: Space Exploration).
- **Mission, Vision &amp; Core Values** — organizational mission, vision,
  seven core values, and six strategic goals.
